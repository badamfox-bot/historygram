Free · Open Source · Runs Locally

See your browsing history as a portrait.

Historygram is a browser extension (Chrome and Firefox) that turns 30 days of your browsing history into a fun, shareable image — top sites, busiest hours, and a personality tag like "Night Owl Builder" or "Early Bird Scroller."

Internal comparison tool — picking a visual direction, will be removed later.

Every category has its own mascot, colors, and title. This is example data — install the extension to generate your real one.

Your data never leaves your device

Websites can't normally read your browser history — that's a deliberate security boundary every modern browser enforces. Historygram works differently: as an installed extension, it asks for your explicit permission, reads your history locally, and renders the image entirely in your browser. No servers, no uploads, no tracking.

Install it (not yet store-published)

Chrome:

  1. Clone or download the GitHub repo
  2. Open chrome://extensions
  3. Turn on Developer mode (top right)
  4. Click Load unpacked and select the extension/ folder
  5. Click the Historygram icon, then Generate My Portrait

Firefox:

  1. Clone or download the GitHub repo
  2. Open about:debugging#/runtime/this-firefox
  3. Click Load Temporary Add-on… and select extension/manifest.json
  4. Click the Historygram icon, then Generate My Portrait
